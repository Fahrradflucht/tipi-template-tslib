# tipi-template-tslib

[![npm][0]][1]
[![Commitizen friendly][2]][3]

This is a [tipi-cli][4] template.

## Usage

```sh
npm install --global tipi-cli
```

```sh
tipi --template=tslib new-project
```

## Development

Installing the dev dependencies:

```sh
npm install
```

Committing a new change:

```sh
npm run commit
```

Releasing a new version:

```sh
npm run release
```
